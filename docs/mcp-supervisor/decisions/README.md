# `@platformos/platformos-mcp-supervisor` architectural decisions

This is the home for decisions made about `@platformos/platformos-mcp-supervisor`'s architecture.

## Contributing a new decision

1. Copy `ADR_TEMPLATE.md` to a new directory in this folder, with a concise
   slug starting with the next sequential number, e.g.
   `002-using-vite/README.md`.
2. Fill in the template.
3. Open a PR.
4. When it's approved, you can merge it.
