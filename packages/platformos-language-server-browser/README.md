# `@platformos/platformos-language-server-browser`

This is the Browser wrapper of the runtime-agnostic [`@platformos/platformos-language-server-common`](https://npm.im/@platformos/platformos-language-server-common) package. It does not come with batteries included and assumes you know what you're doing.

Presumably for use in CodeMirror along with [`@platformos/codemirror-language-client`](https://npm.im/@platformos/codemirror-language-client).

See the [@platformos/platformos-language-server-common README](../theme-language-server-common) for more details.
